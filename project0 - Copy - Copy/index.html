Computer Science Workbook

Python

This site includes materials that were presented in teachers' forums, found on the Internet, and mostly were created by me. All the materials correspond the 2019-2020 8th grade Computer Science curriculum.
The main core of the learning process is projects, each presented in a separate unit. Each unit is divided to several lessons that deal with different aspects required in order to complete the project. By completing these projects, the student is exposed to different paradimes used in the field of Computer Science in general, and by Python in particular.
